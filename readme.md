## TYPESCRIPT

## 需要安装的依赖

> [ts-node](<[https://](https://www.npmjs.com/package/ts-node)>)

```bash
npm i ts-node -g
or
yarn global add ts-node
```

> [ts-node-dev](https://www.npmjs.com/package/ts-node-dev)

```bash
npm i ts-node-dev -g
or
yarn global add ts-node

```

> [typescript](https://www.npmjs.com/package/typescript)

```bash
npm i typescript -g
or
yarn global add typescript
```
